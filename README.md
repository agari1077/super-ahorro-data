# Super-Ahorro - datos de precios (Roca)

Paquete diario de precios de General Roca (4 cadenas SEPA) para la app Super-Ahorro.

- dist/latest.json: metadata (frescura, hash, URL del paquete).
- dist/paquete-<fecha>.json.gz: catalogo de precios.

Datos: Precios Claros / SEPA del gobierno (CC-BY 4.0) - republicacion de datos publicos.
